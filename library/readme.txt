## Spectral Library (MSP Database)

The `library/` folder contains reference MS/MS spectral libraries used for MSP annotation and RT workflow support.

Two public lipid spectral libraries are included:

- MSDIAL Tandem Mass Spectral Atlas VS69 (Negative mode)
- MSDIAL Tandem Mass Spectral Atlas VS69 (Positive mode)

These libraries are publicly available and can be downloaded from:
- MS-DIAL official repository / software distribution
- GNPS (Global Natural Products Social Molecular Networking) platform

Original file names:
- MSDIAL-TandemMassSpectralAtlas-VS69-Neg.msp
- MSDIAL-TandemMassSpectralAtlas-VS69-Pos.msp

These resources are used for lipid spectral matching and are fully reproducible from publicly available databases.
